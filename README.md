## Getting started

Prerequisites
- Git
- Node.js >= 14 (or your project's runtime)
- Package manager: npm or yarn
- Database (if applicable)

Quick start
1. Clone the repository
   git clone https://github.com/bts-singh/trend-market-place.git
2. Change into the project directory
   cd trend-market-place
3. Install dependencies
   npm install
   # or
   yarn install
4. Create / configure environment variables
   - Copy `.env.example` to `.env` and set values (if present)
5. Start the development server
   npm run dev
   # or
   yarn dev

Adjust commands above to match the actual scripts in package.json or your project manifest.
